How Detection Works

This tool analyzes your image to find the pixel scale factor—how much the original pixel art was enlarged. It tests common scales (2×, 3×, 4×, 5×, 6×, 8×, 10×, 12×, 16×) and scores each one based on uniformity: how many pixels in uniform square blocks exist at that scale.

A higher score means the scale is likely correct. If the score is low, your image may not be uniform pixel art, or the true scale wasn't tested. Use the manual override to try different scales.

Once you set the scale factor, the corrected image is generated using nearest-neighbor sampling— taking exactly 1 pixel from each block. No blurring, interpolation, or color averaging is applied.
